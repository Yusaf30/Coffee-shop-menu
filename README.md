# Brew & Bean Coffee Shop Website ☕

**Brew & Bean** is a simple and elegant coffee shop website created using **HTML and CSS**.
The website showcases a café-style menu layout with different coffee drinks, featured items, and information about the coffee shop.

This project focuses on creating a **clean, structured, and visually appealing layout** inspired by modern café websites.

---

## ✨ Features

* Hero header section with coffee background image
* Signature drinks menu section
* Classic coffee favorites section
* Today's special menu section
* Coffee shop story section
* Card-based menu layout
* Clean café-style design

---

## 🛠️ Technologies Used

* **HTML5** – for structuring the webpage
* **CSS3** – for layout and styling

---

## 📑 Website Sections

1. **Hero Section** – Introduction to Brew & Bean coffee shop
2. **Signature Drinks** – Featured coffee drinks
3. **Classic Favorites** – Popular coffee menu items
4. **Our Story** – Background and concept of the café
5. **Today's Special** – Highlighted seasonal drinks and snacks
6. **Visit Us Today** – Café address and opening hours

---

## 🎯 Purpose of the Project

The main purpose of this project is to practice:

* HTML webpage structure
* CSS layout and styling
* Creating card-based menu sections
* Designing a themed café website
